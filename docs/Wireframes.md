## スマホ向け棚卸アプリ ワイヤーフレーム（簡易モック）

注: 等幅フォント前提のテキストモックです。実装ではモバイル幅（~390px）想定の縦長UI。

### 1) ダッシュボード
```
+--------------------------------------------------+
|  ヘッダー：棚卸                                     |
+--------------------------------------------------+
|  [スキャンを開始]   [CSVエクスポート]               |
|                                                  |
|  ┌ カード ────────────────┐   ┌ カード ─────────┐ |
|  │ 総型番数:  12                        │   │ 合計数量:  245     │ |
|  └──────────────────────────┘   └─────────────────┘ |
|                                                  |
+--------------------------------------------------+
|  フッター：ナビ（スキャン｜一覧｜設定｜履歴）        |
+--------------------------------------------------+
```

### 2) スキャン
```
+--------------------------------------------------+
|  ヘッダー：スキャン                                  |
+--------------------------------------------------+
|  [ video 背面カメラプレビュー ]                     |
|  ┌───────── ガイド枠 ─────────┐             |
|  │                                     │             |
|  └─────────────────────────────────────┘             |
|   状態: 読み取り中...                                 |
|   トースト(成功時): AB-123 を+1カウント               |
|   トースト(失敗時): デコードに失敗しました            |
|                                                      |
|  [停止/再開]   [カメラ切替]                           |
+--------------------------------------------------+
|  フッター：ナビ（スキャン｜一覧｜設定｜履歴）        |
+--------------------------------------------------+
```

### 3) 一覧（集計）
```
+--------------------------------------------------+
|  ヘッダー：一覧                                      |
+--------------------------------------------------+
|  型番（昇順）                 数量        操作         |
|  -----------------------------------------------   |
|  AB-123                        10        [-]  [+]   |
|  AB-124                         0        [-]  [+]   |
|  ZX-999                         7        [-]  [+]   |
|  ...                                               |
|  （更新直後の行は一時ハイライト）                     |
|                                                  |
|  [型番追加]  [CSV出力]  [全削除]                    |
+--------------------------------------------------+
|  フッター：ナビ（スキャン｜一覧｜設定｜履歴）        |
+--------------------------------------------------+
```

### 4) 設定
```
+--------------------------------------------------+
|  ヘッダー：設定                                      |
+--------------------------------------------------+
|  抽出                                              |
|   開始位置N: [ 49 ]  (0〜500)                       |
|  スキャン挙動                                       |
|   クールダウン(ms): [ 1800 ] (50〜2000)              |
|   ビープ音: [ON]   バイブ: [ON]                      |
|                                                      |
|  [保存]  [設定のリセット]  [データ全消去]            |
+--------------------------------------------------+
|  フッター：ナビ（スキャン｜一覧｜設定｜履歴）        |
+--------------------------------------------------+
```

### 5) 履歴
```
+--------------------------------------------------+
|  ヘッダー：履歴                                      |
+--------------------------------------------------+
|  最新順                                             |
|  10:30  AB-123   +1   RAW: ...ABCDEFG...(先頭64)     |
|  10:29  ZX-999   -1   RAW: ""                         |
|  ...                                               |
|                                                     |
|  [選択削除]   [全削除]（確認ダイアログ）              |
+--------------------------------------------------+
|  フッター：ナビ（スキャン｜一覧｜設定｜履歴）        |
+--------------------------------------------------+
```
